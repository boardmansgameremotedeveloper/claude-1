# [Your Store Name]

A modern e-commerce storefront built with Shopify Hydrogen.

## Tech Stack

- Hydrogen (Remix-based framework)
- Shopify Storefront API
- TypeScript
- Tailwind CSS

## Getting Started
```bash
npm install
npm run dev
```

Visit `http://localhost:3000`

## Environment Variables

Copy `.env.example` to `.env` and configure:
- `PUBLIC_STOREFRONT_API_TOKEN`
- `PUBLIC_STORE_DOMAIN`

## Project Structure

- `/app/routes` - Page routes
- `/app/components` - Reusable components
- `/app/styles` - Global styles
